## Purpose

Government document management and operations system for managing:

- **Incoming Documents** (Văn bản đến)
- **Outgoing Documents** (Văn bản đi)
- **Internal Documents** (Văn bản nội bộ)
- **Work Tasks & Assignments** (Công việc và phân công)
- **Workflow Processes** (Quy trình làm việc)
- **Digital Signatures** (Chữ ký số)

---

## Key Features

### Document Management

- Full document lifecycle (create, review, approve, distribute)
- Document tracking and history
- Document retake (thu hồi)
- Search and filtering

### Task Management

- Task creation and assignment
- Progress tracking
- Delegation support
- Two versions: legacy (`task/`) and new (`task-v2/`)

### Workflow

- BPMN-based workflow configuration
- Approval flows
- Delegation flows

### Security

- JWT authentication with refresh tokens
- USB Token (VGCA) hardware authentication
- Document encryption
- Role-based access control

### Additional

- Work calendar management
- Daily reports
- Vehicle management
- Organization hierarchy
- User profiles and settings
